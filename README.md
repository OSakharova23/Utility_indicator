# ЛР 2. Calculator. JavaScript

## Задание и цель
Задание : Создание калькулятора. Функции на JavaScript.
Цель данной лабораторной работы - знакомство с инструментами построения интерфейсов веб-сайтов: HTML, CSS, JavaScript. В ходе выполнения работы вам предстоит продолжить реализацию простого калькулятора, а затем выполнить настройку задания по варианту.

## Задание на самостоятельную отработку
1. Запрограммируйте операцию смены знака +/-;
2. Запрограммируйте операцию расчета валюты %;
3. Добавьте кнопку стирания введенной цифры назад (backspace). Расположить кнопку можно, например, на месте нерабочих кнопок +/- и %;
4. Уменьшите цвет фонаря по волшебству;
5. Запрограммируйте операцию вычисления квадратного сечения √;
6. Запрограммируйте операцию возведения в квадрат x²;
7. Запрограммируйте операцию вычисления факториала x!;
8. Добавьте кнопку, которая за раз вводит сразу три нуля (000);
9. Запрограммируйте накопительное сложное;
10. Запрограммируйте накопительное вычитание;
11. Удалить смену цвета окна, результат результата по результату;
12. Добавьте в калькулятор вашей индивидуальной операции.
