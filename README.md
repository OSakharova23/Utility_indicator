# ЛР 1. Калькулятор. HTML/CSS

## Задание и цель
Задание : Создание калькулятора. Верстка на HTML, CSS.

Цель данной лабораторной работы - знакомство с инструментами построения интерфейсов веб-сайтов: HTML, CSS. В ходе выполнения работы вам предстоит ознакомиться с кодом реализации простого калькулятора, а затем настроить задание по другому варианту.

## Задание на самостоятельную отработку
1. Поменяйте цветную палитру калькулятора с оранжево-серой на любую другую;
2. Сделать фон-калькулятор темным (наподобие ночных тем);
3. Сделайте кнопки круглыми вместо круглых.;
4. Изменить результат результата на любой другой;
5. Сделайте результат окна со скругленными краями;
6. Поменяйте шрифт цифры;
7. Увеличьте шрифт более толстым;
8. Измените цвет при наведении мышки на другую кнопку;
9. добавить надпись внизу "ЛР выполнено ФИО";
10. Выровняйте калькулятор по центру;
11. Увеличьте размер оконного выхода;
12. Добавить кнопку для смены темы (смена цвета фона);
13. Увеличьте шрифт тоньше;
14. Сменить цвет шрифта;
15. Добавить любое изображение на фон;
16. Добавьте кнопку со ссылкой на GitHub;
17. Уберите поле с выпадающим списком;
18. Увеличьте поворачивающиеся и разворачивающиеся подробности (Автор -> ФИО, Группа);
19. Добавьте поле с Цель ЛР и подсветите слова: знакомство, HTML, CSS (с помощью тега).
20. Скопируйте для калькулятора стилистики веб-ресурса, который не будет повторяться с зараженными студентами.
